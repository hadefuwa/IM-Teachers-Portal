## TIA Portal Project

- `smart-factory-tia-portal.zip` - Complete Siemens TIA Portal project

## File Preparation

### TIA Portal Project
1. Export your complete TIA Portal project
2. Include all necessary files:
   - Project files (.ap17, .ap18, etc.)
   - Libraries and symbols
   - HMI graphics and screens
   - Tag definitions
3. Add a README.txt with installation instructions
4. Compress everything into `smart-factory-tia-portal.zip`

## File Size Considerations

Large files (>100MB) may not work well with GitHub Pages. Consider:
- Using GitHub Releases for large downloads
- Hosting files on cloud storage (Google Drive, Dropbox, etc.)
- Using a CDN or file hosting service

Update the download links in `js/main.js` accordingly.
